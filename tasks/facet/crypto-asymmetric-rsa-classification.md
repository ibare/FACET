# RSA 분류

## 1. 대상 정보

- 이름: RSA
- 분야: 보안·암호 / 비대칭 암호 (공개키 암호)
- 핵심 단계: 키 생성 (두 소수로부터 공개키·비밀키 짝 생성), 공개키 암호화, 비밀키 복호화
- 본질: 두 키의 비대칭성, 큰 수 인수분해의 어려움에 기반한 일방향성
- 범위 밖: 디지털 서명, 패딩 (OAEP/PSS), 측면 채널 공격, 후양자 암호

## 2. type

algorithm

## 3. 진행 모델

시간 진행형

## 4. 핵심 학습 질문

두 소수로부터 만든 공개키와 비밀키가 어떻게 한쪽으로만 잠그고 다른 쪽으로만 풀 수 있는 비대칭 짝이 되며, 그 비대칭성은 왜 큰 수 인수분해의 어려움에 의지하는가?
