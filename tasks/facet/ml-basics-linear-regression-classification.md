# 선형 회귀 분류

## 1. 대상 정보

- 이름: 선형 회귀 (linear regression)
- 분야: 머신러닝 기초 / 지도학습
- 도메인 prefix: `ml-basics` — 기존 `datastructure-`, `messaging-` 처럼 짧고 분야 식별이 명확. `machine-learning` 은 너무 광범위하고, `supervised-learning` 은 분류/회귀 전부를 포괄해 facet 한 건의 슬러그로는 길다. 향후 분류·군집 등 다른 ML 도메인이 늘어나면 별도 prefix 분리도 고려.
- 핵심 입출력: 입력 x (특징) 와 정답 y (스칼라) 의 점들을 받아 직선 y ≈ wx + b 의 매개변수 w, b 를 출력.
- 핵심 사건/연산: 데이터 점 분포 확인 → 초기 직선 (w, b) 설정 → 각 점의 잔차 계산 → 평균제곱오차(MSE) 손실 산출 → 손실의 기울기로 w, b 갱신 (경사하강) → 반복하며 수렴.

## 2. type

algorithm

> 학습 운동 (반복 갱신) 자체를 보여주는 시각화이므로, 정적 모델 정의(`concept`) 가 아닌 경사하강 기반 학습 알고리즘으로 잡는다.

## 3. 진행 모델

시간 진행형

> 경사하강 반복(iteration) 이 시간 축 위에서 펼쳐지며 직선이 점차 잔차의 제곱합을 줄이는 운동을 보여주는 것이 학습 핵심. 점 추가 같은 입력 반응형 요소는 보조 인터랙션으로만 다룬다.

## 4. 핵심 학습 질문

데이터 점들의 분포 위에서 잔차의 제곱합을 최소화하는 직선이 경사하강 반복을 통해 어떻게 점진적으로 발견되는가?
