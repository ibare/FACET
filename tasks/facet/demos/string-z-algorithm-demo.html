Z-알고리즘이란? 문자열의 각 위치 i 에서 “처음과 얼마나 겹치는지” Z[i] 를 한 번의 좌→우 스캔으로 모두 구합니다. 지금까지 발견한 가장 오른쪽 매치 구간 [l, r] 을 박스로 기억하고, 새 위치 i 가 박스 안이면 거울 위치 k = i - l 의 답을 빌려 비교를 건너뜁니다. r 이 절대 줄지 않아 전체가 O(n) 입니다. 이 Z 배열로 패턴 P 를 텍스트 T 에서 찾을 때는 P + 구분자 + T 의 Z 배열에서 |P| 와 같은 값이 나오는 위치를 보면 됩니다.
시작 — ▶ 버튼을 눌러 진행하세요.
읽는 법
세로 막대 = 확정된 Z[i]. 박스 안 영역과 좌측 prefix 영역은 같은 색이며, 박스 안에 i 가 들어오면 거울 호가 떠올라 k = i − l 의 답을 빌려옵니다.
실제 비교 0
Naive 예상 ≈ 0
l ← 0, r ← 0
for i ← 1 .. n−1:
  if i < r: Z[i] ← min(r − i, Z[i − l]) ← 거울 빌리기
  while i + Z[i] < n and S[Z[i]] = S[i + Z[i]]:
    Z[i] ← Z[i] + 1 ← 실제 비교
  if i + Z[i] > r: l ← i, r ← i + Z[i] ← 박스 우측 확장

핵심 관찰: ① 박스 안에서 떠오르는 거울 호와 ② 호가 도착하는 순간 i 위치가 한 번에 연두로 칠해지는 면적, 그리고 ③ 우상단 누적 카운터가 Naive 예상치보다 한참 적게 자라는 모습입니다.
