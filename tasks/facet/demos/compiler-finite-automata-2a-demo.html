대상 개념 — 유한오토마타

유한오토마타는 입력 문자열을 한 글자씩 읽으면서 유한 개의 상태 사이를 옮겨 다니다가, 입력이 끝났을 때 어디에 있는지로 받아들일지 거절할지를 결정하는 단순한 인식기다. 갈 길이 항상 한 곳뿐이면 결정적(DFA), 같은 글자에 갈 곳이 여러 곳일 수 있으면 비결정적(NFA)이다. 컴파일러의 렉서는 이 메커니즘으로 소스 코드 글자 스트림에서 식별자/숫자/키워드 같은 토큰을 잘라낸다.
"한 글자씩 먹으며 상태 사이를 옮겨 다니다, 다 먹었을 때 받아들일지 거절할지를 결정하는 작은 기계."
